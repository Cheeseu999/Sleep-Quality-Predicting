# Sleep Duration Predicting
Project to predict a person's average amount of sleep based on their lifestyle characteristics. The goal is to get a better understanding of what qualities of a person's life may be affecting the duration that they are sleeping.

**File Contents**
- "Sleep_Quality_Code.Rmd" R Markdown file.
- "Sleep_Quality_Presentation.pdf" Accompanying Presentation.
- README.md 

The dataset is sourced from [Kaggle](https://www.kaggle.com/datasets/uom190346a/sleep-health-and-lifestyle-dataset). A Linear Regression model was trained onto the dataset. The R Markdown files contains all preprocessing, training, and model selecting while the presentation summarizes all key findings.
